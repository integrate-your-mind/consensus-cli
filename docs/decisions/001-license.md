# 001 - License choice

## Problem
Pick a license that is permissive and supports commercial adoption.

## Options
- MIT
- Apache-2.0
- GPLv3

## Decision
Apache-2.0.

## Tradeoffs
- Apache-2.0 is permissive with an explicit patent grant, but longer text.

## Revisit
If we ever want strong copyleft, revisit GPL/AGPL options.
