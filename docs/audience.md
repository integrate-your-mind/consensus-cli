# Audience

## Primary user
Developers running multiple Codex sessions locally.

## Primary customer
Individual developers and small teams that pay for hosted coordination later.

## Top user goals
1) See active agents at a glance.
2) Understand what each agent is doing without digging through logs.
3) Spot failures quickly.

## Top user fears
1) Leaking sensitive data.
2) Losing context across agents.
3) Tooling that is hard to set up or maintain.
