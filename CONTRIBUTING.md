# Contributing

Thanks for helping improve consensus.

## Quick start
- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`

## Workflow
- Create a branch from main.
- Keep changes small and focused.
- Update docs when behavior or config changes.
- Ensure `npm run build` passes.

## UI tests
- Install browsers: `npx playwright install`
- Run: `npm run test:ui`

## Code of Conduct
By participating, you agree to follow `CODE_OF_CONDUCT.md`.

## Reporting issues
- Use GitHub Issues in this repo.
- Include repro steps, logs, and expected behavior.

## Pull requests
Please include:
- What changed and why.
- How it was tested.
- Docs updates (if applicable).
- Notes about breaking changes (if any).
